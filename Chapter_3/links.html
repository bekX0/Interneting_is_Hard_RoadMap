There are three types of links:

  • Absolute Links
  • Relative Links
  • Root-Relative Links

Absolute Links

Refers to external link that independent from base website
They start with a scheme ('http://' or 'https://'), followed by the domain name of the website, then the path of target webpage

Ex: https:// developer.mozilla.org /en-us/docs...
